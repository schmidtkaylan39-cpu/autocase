# Release v0.1.4

Release date: 2026-04-20 (Asia/Shanghai)

## Release Focus

This promotion stabilizes the autonomous live-acceptance path and makes release review evidence easier to hand off cleanly.

- keeps the default GPT-5.4 + Codex + local-ci route intact
- hardens degraded fallback, orphaned-lock recovery, and no-progress diagnostics
- adds a human-friendly Chinese local panel and a consistent GPT review-share package flow

## Key Changes

- added a Chinese-first local operator panel for common workflow actions:
  - one-click run setup
  - clearer intake guidance
  - direct GPT prompt evidence viewing
- strengthened autonomous and dispatch recovery behavior:
  - reclaim dead execution locks promptly
  - separate reviewer/verifier replan budgets from implementation budget
  - terminate degraded no-progress loops with diagnosable artifacts instead of silent long stalls
  - cap launcher output buffering while preserving useful stderr tails
- tightened acceptance and evidence packaging:
  - stronger live-acceptance tests around planner preflight, degraded fallback, and orphaned execution locks
  - stricter validation-gate and E2E evidence coverage
  - added `npm run review:share` to package prompt, brief, validation, bundle, and acceptance evidence from one snapshot

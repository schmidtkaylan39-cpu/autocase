## Release Focus

This promotion is a hardening release on top of the GPT-5.4 + Codex baseline.

- keeps the default route unchanged (`manual` planner/reviewer + `codex` executor + `local-ci` verifier)
- promotes workflow and review-bundle guardrails discovered in final external review rounds
- targets release reliability and reviewer rerun clarity, not feature expansion

## Key Changes

- review-bundle brief now highlights rerun prerequisite near the top:
  - bundle is source snapshot + retained evidence
  - rerun repo-level validations only after `npm ci` from `repo/`
- workflow validation now includes semantic checks for release-readiness:
  - `.github/workflows/release-readiness.yml` must include Windows `backup:project` smoke
  - `.github/workflows/release-readiness.yml` must include Windows `release:win` smoke

- added focused workflow semantic tests to lock those guarantees
- updated release-readiness docs to explicitly describe the new workflow semantic guard

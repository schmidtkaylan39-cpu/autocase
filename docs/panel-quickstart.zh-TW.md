# AI Factory 面板使用說明（繁中）

這個面板是給一般使用者操作的，不需要會寫程式。

## 1. 啟動面板

在專案根目錄開 PowerShell，執行：

```bash
node src/index.mjs panel .
```

瀏覽器打開：

```text
http://127.0.0.1:4310
```

如果 `4310` 被占用，換埠號：

```bash
node src/index.mjs panel . 4320
```

## 2. 最簡單操作（推薦）

面板上只要做三件事：

1. 確認「工作區路徑」
2. 輸入「需求內容」
3. 按「一鍵開始（推薦）」

一鍵開始會自動依序執行：

1. 初始化
2. 需求澄清（intake）
3. 需求確認（confirm）
4. 建立 Run
5. 全自動執行（autonomous）

## 3. 怎麼看有沒有成功

看「目前狀態」區塊：

- 需求狀態：`已確認`
- 執行狀態：`已完成`
- 阻塞任務：`0`
- 失敗任務：`0`

再看「操作紀錄」最後一條是否顯示：

- `一鍵開始完成`

## 4. 輸出檔案位置

假設工作區是 `C:\demo`，會產生：

- `C:\demo\runs\<run-id>\run-state.json`
- `C:\demo\runs\<run-id>\report.md`
- `C:\demo\runs\<run-id>\autonomous-summary.json`
- `C:\demo\runs\<run-id>\autonomous-summary.md`

## 5. 常見情況

- 一鍵開始失敗，顯示「確認失敗（需補資訊）」  
  需求太模糊，請把需求改得更具體（輸入檔、輸出檔、不要做什麼），再按一次。

- 執行狀態不是「已完成」  
  先按「重新整理狀態」，再看「操作紀錄」最後一條錯誤內容。

- 想手動一步一步跑  
  打開「進階操作（選用）」再按對應按鈕。

## 6. 關閉面板

回到啟動面板的終端視窗，按：

```text
Ctrl + C
```
